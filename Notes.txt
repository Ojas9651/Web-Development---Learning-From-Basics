There are 3 ways to pply CSS -

Inline
Internal
External

Inline - <h1 style="background-color: aqua">Hello World!</h1>

Internal -

<head>
	<style>
		h1{
			background-color: blue;
		}
	</style>
</head>

Enternal -

<head>
	<link rel="stylesheet" href="">
</head>

What are selectors ?
This is how we select an element to apply CSS.

Selecting by tag -

h1{
}

Selcting by ID -

#id_name{
}

Selecting by Class -

.class_name{
}

Group Selector -

h1, id {
    background-color: aqua;
}
